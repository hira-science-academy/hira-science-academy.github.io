Unit 3: Sets and Functions

Class 9 Mathematics | Punjab Board Syllabus 2025

1. Define set.

A collection of well-defined distinct objects is called a set. It is denoted by capital letters \( A, B, C \), etc.

Example: \( A = \{1, 2, 3, 4\} \)

2. How can a set be described using different methods?

A set can be described using three different methods:

  1. Descriptive Form: A set is described in words without listing its elements.

    Example: The set of all vowels in the English alphabet.

  2. Tabular Form (Roster Form): A set is described by listing its elements within curly brackets \(\{\}\).

    Example: \( A = \{a, e, i, o, u\} \)

  3. Set-Builder Form: A set is described by stating a property that all its elements share.

    Example: \( A = \{x \mid x \text{ is a vowel of the English alphabet}\} \)

Common Sets in Algebra:

  • \( \mathbb{N} = \{1, 2, 3, \ldots\} \) (Natural numbers)
  • \( \mathbb{W} = \{0, 1, 2, 3, \ldots\} \) (Whole numbers)
  • \( \mathbb{Z} = \{0, \pm1, \pm2, \pm3, \ldots\} \) (Integers)
  • \( \mathbb{Q} = \left\{\frac{p}{q} \mid p, q \in \mathbb{Z}, q \neq 0\right\} \) (Rational numbers)
  • \( \mathbb{R} = \mathbb{Q} \cup \mathbb{Q}' \) (Real numbers)

3. Define singleton set.

A set with only one element is called a singleton set.

Examples: \( \{3\}, \{a\}, \{\text{Saturday}\} \)

4. Define empty set.

The set with no elements is called an empty set, null set, or void set. It is denoted by \( \varnothing \) or \(\{\}\).

5. Define equal sets.

Two sets \( A \) and \( B \) are equal if they have exactly the same elements.

Example: \( \{1, 2, 3\} \) and \( \{2, 1, 3\} \) are equal sets.

6. Define equivalent sets.

Two sets \( A \) and \( B \) are equivalent if they have the same number of elements, denoted by \( A \sim B \).

Example: \( A = \{a, b, c, d, e\} \) and \( B = \{1, 2, 3, 4, 5\} \) are equivalent sets.

7. Define subset.

If every element of set \( A \) is an element of set \( B \), then \( A \) is a subset of \( B \), denoted by \( A \subseteq B \).

Note: \( B \) is called a superset of \( A \), denoted by \( B \supseteq A \).

8. Define proper subset.

If \( A \subseteq B \) and \( B \) contains at least one element not in \( A \), then \( A \) is a proper subset of \( B \), denoted by \( A \subset B \).

9. Define improper subset.

If \( A \subseteq B \) and \( A = B \), then \( A \) is an improper subset of \( B \). Every set is an improper subset of itself.

10. Define universal set.

The set that contains all objects or elements under consideration is called the universal set, denoted by \( U \).

11. Define power set.

The power set of a set \( S \), denoted by \( P(S) \), is the set of all possible subsets of \( S \). If \( S \) has \( m \) elements, then \( P(S) \) has \( 2^m \) elements.

Example: If \( S = \{1, 2\} \), then \( P(S) = \{\varnothing, \{1\}, \{2\}, \{1, 2\}\} \).

12. Define union of sets.

The union of sets \( A \) and \( B \), denoted by \( A \cup B \), is the set of all elements that belong to \( A \) or \( B \).

Example: If \( A = \{1, 2\} \) and \( B = \{1, 3\} \), then \( A \cup B = \{1, 2, 3\} \).

13. Define intersection of sets.

The intersection of sets \( A \) and \( B \), denoted by \( A \cap B \), is the set of all elements that belong to both \( A \) and \( B \).

Example: If \( A = \{1, 2\} \) and \( B = \{1, 3\} \), then \( A \cap B = \{1\} \).

14. Define difference of sets.

The difference \( A - B \) is the set of elements in \( A \) that are not in \( B \). Similarly, \( B - A \) is the set of elements in \( B \) that are not in \( A \).

Example: If \( A = \{1, 2\} \) and \( B = \{1, 3\} \), then \( A - B = \{2\} \) and \( B - A = \{3\} \).

15. Define complement of a set.

The complement of set \( A \), denoted by \( A' \) or \( A^c \), is the set of all elements in the universal set \( U \) that are not in \( A \).

Example: If \( U = \{1, 2, 3, 4, 5\} \) and \( A = \{1, 3\} \), then \( A' = \{2, 4, 5\} \).

16. What are disjoint sets?

Two sets are disjoint if their intersection is the empty set.

Example: The set of odd natural numbers and the set of even natural numbers are disjoint.

17. What are overlapping sets?

Two sets are overlapping if their intersection is non-empty but neither is a subset of the other.

Example: \( L = \{2, 3, 4, 5, 6, 7, 8, 9\} \) and \( M = \{5, 6, 7, 8, 9, 10\} \) are overlapping sets.

18. Define Venn diagram.

A Venn diagram uses rectangles and circles to visually represent relationships between sets. The universal set is represented by a rectangle, and its subsets by circles inside the rectangle.

19. What is the cardinality of a set?

The cardinality of a set is the number of elements in the set, denoted by \( n(A) \).

Example: If \( A = \{1, 3, 5, 7, 9, 11\} \), then \( n(A) = 6 \).

20. What is the Inclusion-Exclusion Principle?

The Inclusion-Exclusion Principle calculates the number of elements in the union of two or more sets by avoiding overcounting.

For Two Sets: \( n(A \cup B) = n(A) + n(B) - n(A \cap B) \)

For Three Sets: \( n(A \cup B \cup C) = n(A) + n(B) + n(C) - n(A \cap B) - n(A \cap C) - n(B \cap C) + n(A \cap B \cap C) \)

21. What is a relation in mathematics?

A relation is any set of ordered pairs. The relationship between the components may or may not be specified.

Examples:

  • Greater than: \( (5, 4) \)
  • Square: \( (25, 5) \)
  • Equal: \( (2 \times 2, 4) \)

22. Define Cartesian product.

The Cartesian product of sets \( A \) and \( B \), denoted by \( A \times B \), is the set of all ordered pairs \( (x, y) \) where \( x \in A \) and \( y \in B \).

Example: If \( A = \{1, 2\} \) and \( B = \{3, 4\} \), then \( A \times B = \{(1, 3), (1, 4), (2, 3), (2, 4)\} \).

23. Define binary relation.

A binary relation is any subset of the Cartesian product \( A \times B \). It is usually denoted by \( r \).

24. What are the domain and range of a relation?

Domain: The set of first elements of the ordered pairs in a relation.

Range: The set of second elements of the ordered pairs in a relation.

Example: For \( r = \{(1, 2), (1, 3), (2, 3)\} \), the domain is \( \{1, 2\} \) and the range is \( \{2, 3\} \).

25. What is a relation on a set?

A relation on a set \( A \) is any subset of \( A \times A \).

26. Define function.

A function \( f: A \to B \) is a relation where:

  1. \( f \subseteq A \times B \),
  2. Every element of \( A \) is the first element of exactly one ordered pair in \( f \).

Example: For \( A = \{0, 1, 2, 3, 4\} \) and \( B = \{3, 5, 7, 9, 11\} \), define \( f = \{(x, y) \mid y = 2x + 3\} \).

Then \( f = \{(0, 3), (1, 5), (2, 7), (3, 9), (4, 11)\} \).

Domain: \( A \), Range: \( \{3, 5, 7, 9, 11\} \).

27. What is meant by domain and co-domain of a function?

Domain: The set of all possible input values (first elements) of the function.

Co-domain: The set of all possible output values (second elements) of the function.

28. What is an Into Function?

An into function \( f: A \to B \) is one where the range is a proper subset of the co-domain \( B \) (i.e., \( \text{Range } f \subset B \)).

29. What is a One-One Function (Injective Function)?

A function \( f: A \to B \) is injective if no two distinct elements of \( A \) map to the same element in \( B \).

30. What is an Onto Function (Surjective Function)?

A function \( f: A \to B \) is surjective if every element of \( B \) is mapped to by some element of \( A \) (i.e., \( \text{Range } f = B \)).

31. What is a Bijective Function?

A function \( f: A \to B \) is bijective if it is both injective and surjective (one-one and onto).

32. What is the Notation of a Function?

Functions can be described using set-builder notation, especially for infinite sets.

Example: \( f = \{(x, y) \mid y = x^2, x \in \mathbb{N}\} \).

33. Define a linear function.

A linear function has the form \( \{(x, y) \mid y = mx + c\} \), where \( m \) is the slope and \( c \) is the y-intercept.

34. What is the general form of a quadratic function?

A quadratic function has the form \( \{(x, y) \mid y = ax^2 + bx + c\} \), representing a parabola.
