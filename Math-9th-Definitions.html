Mathematics Class 9 Definitions

This comprehensive collection contains all key definitions, terms, formulas, and concepts from the Class 9 Mathematics syllabus according to the Punjab Textbook Board. Each chapter is organized systematically with clear explanations and examples.

How to Use This Resource:

  • Click on any chapter in the sidebar to jump directly to its definitions
  • Each term is highlighted for easy reference
  • Important formulas are boxed for quick identification
  • For detailed explanations, visit the individual chapter pages

Chapter 1: Real Numbers

View Full Chapter

Real Numbers

The set of all rational and irrational numbers. Represented by ℝ.

Rational Numbers

Numbers that can be expressed as a fraction p/q where p and q are integers and q ≠ 0. They have terminating or repeating decimal expansions.

Irrational Numbers

Numbers that cannot be expressed as simple fractions. Their decimal expansions are non-terminating and non-repeating (e.g., √2, π).

Radical Properties:
• √(ab) = √a × √b
• √(a/b) = √a/√b
• (√a)² = a

Chapter 2: Logarithms

View Full Chapter

Logarithm

The exponent to which a base must be raised to produce a given number. If aˣ = N, then logₐN = x.

Laws of Logarithms:
• logₐ(mn) = logₐm + logₐn
• logₐ(m/n) = logₐm - logₐn
• logₐ(mⁿ) = n logₐm
• logₐa = 1
• logₐ1 = 0

Common Logarithm

A logarithm with base 10, written as log N (base 10 is implied).

Chapter 3: Sets and Functions

View Full Chapter

Set

A well-defined collection of distinct objects, called elements.

Function

A relation between two sets where each element of the first set (domain) is paired with exactly one element of the second set (codomain).

Set Operations:
• Union (A ∪ B): All elements in A or B
• Intersection (A ∩ B): Elements common to A and B
• Complement (A'): Elements not in A
• Difference (A - B): Elements in A but not in B

Chapter 4: Factorization

View Full Chapter

Factorization

The process of breaking down an expression into a product of simpler expressions (factors).

Factorization Formulas:
• a² - b² = (a - b)(a + b)
• a² ± 2ab + b² = (a ± b)²
• a³ ± b³ = (a ± b)(a² ∓ ab + b²)
• x² + (a+b)x + ab = (x + a)(x + b)

Common Factor

A factor that appears in all terms of an expression.

Chapter 5: Linear Equations

View Full Chapter

Linear Equation

An equation of degree 1 that can be written in the form ax + b = 0 where a ≠ 0.

Inequality

A mathematical statement that compares two expressions using >, <, ≥, or ≤.

Solution Rules:
• Add/subtract same value to both sides
• Multiply/divide both sides by positive number
• Reverse inequality when multiplying/dividing by negative

Chapter 6: Trigonometry

View Full Chapter

Trigonometric Ratios

Ratios of sides in a right-angled triangle relative to an acute angle.

Basic Ratios:
• sin θ = Opposite/Hypotenuse
• cos θ = Adjacent/Hypotenuse
• tan θ = Opposite/Adjacent
• cosec θ = 1/sin θ
• sec θ = 1/cos θ
• cot θ = 1/tan θ

Pythagorean Identity

sin²θ + cos²θ = 1

Chapter 7: Coordinate Geometry

View Full Chapter

Cartesian Plane

A two-dimensional plane with perpendicular x and y axes intersecting at the origin (0,0).

Key Formulas:
• Distance: √[(x₂-x₁)² + (y₂-y₁)²]
• Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
• Slope: (y₂-y₁)/(x₂-x₁)

Linear Equation Forms

Slope-intercept: y = mx + c
General form: Ax + By + C = 0

Chapter 8: Logic

View Full Chapter

Proposition

A declarative statement that is either true or false but not both.

Logical Connectives:
• Conjunction (AND): p ∧ q
• Disjunction (OR): p ∨ q
• Negation (NOT): ¬p
• Conditional: p → q
• Biconditional: p ↔ q

Truth Table

A table showing all possible truth values of a compound statement.

Chapter 9: Similar Figures

View Full Chapter

Similar Figures

Figures with the same shape but not necessarily the same size, having corresponding angles equal and sides proportional.

Similarity Criteria:
• AA (Angle-Angle)
• SAS (Side-Angle-Side)
• SSS (Side-Side-Side)
• Ratio of areas = (Ratio of sides)²

Scale Factor

The ratio of corresponding lengths in similar figures.

Chapter 10: Graphs of Functions

View Full Chapter

Linear Graph

A straight line graph representing a linear function y = mx + c.

Quadratic Graph

A parabola-shaped graph representing a quadratic function y = ax² + bx + c.

Graph Features:
• Slope (m): Steepness of linear graph
• Vertex: Turning point of parabola
• Roots: x-intercepts of the graph

Chapter 11: Loci and Construction

View Full Chapter

Locus

The set of all points that satisfy a given condition or set of conditions.

Common Loci:
• Circle: Locus of points at fixed distance from center
• Perpendicular bisector: Locus equidistant from two points
• Angle bisector: Locus equidistant from two lines

Geometric Construction

Creating geometric figures using only compass and straightedge.

Chapter 12: Information Handling

View Full Chapter

Frequency Distribution

A table showing how often each value or range of values occurs in a data set.

Measures of Central Tendency:
• Mean = Σx/n
• Median = Middle value
• Mode = Most frequent value
• Range = Max - Min

Histogram

A bar graph representing frequency distribution of numerical data.

Chapter 13: Probability

View Full Chapter

Probability

A measure of the likelihood that an event will occur, calculated as the ratio of favorable outcomes to total possible outcomes.

Probability Formulas:
• P(E) = n(E)/n(S)
• 0 ≤ P(E) ≤ 1
• P(E') = 1 - P(E)

Sample Space

The set of all possible outcomes of an experiment, denoted by S.
